Easy to Use and User Friendly Cloud Storage Solution

Product Features

  • File Sharing
  • 2FA
  • Redundancy
  • Security

User can input participants' email to allow file access. For user convenience, a unique code will be generated to allow for quick and easy sharing of files.

Our login system uses 2FA to enhance the security to their accounts. When the a request to login is detected by our server, our database will send a generated code to the clientâ€™s email and they need to provide the code to be able to authenticated

All of the file that user uploaded will be stored in our main storage server and our backup storage server, which means the fragment of data that stored in those 2 storage server have same identity. In the unlikely event of a server outage, such as one caused by the storage server is under maintenance, fragment of data in backup storage will still available to be accessed

Our website is enhanced by SSL/TLS connection to make sure all the packets that are transmitted between the client and server are encrypted
